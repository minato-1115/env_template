# React + TypeScript + Vite

## 実験の表紙のテンプレートを作成しました。　

`git clone https://github.com/minato-1115/ReportTemplate.git` を実行してください。

プロジェクト配下とserverディレクトリとclientディレクトで`npm install` を実行してください。

prettierを使う場合は、プロジェクトのディレクトリ配下でも`npm install`を実行してください。

`npm run dev`で起動可能です。
